# Cloner un repository distant

La commande est :

```
git clone <url_repository>
```

Exemple :

```
git clone https://github.com/jlg-formation/git-manual.git
```

Ou pour cloner un repo local :

```
git clone D:/___GIT/server-local/git-manual
```
